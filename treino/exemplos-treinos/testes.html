Dynamic Div Creation

Click on the first div to create and display the second div. This demonstrates how to use JavaScript to dynamically create and manipulate elements in the DOM.

First Div

Click me to create the second div!

How it works:

// Get references to the DOM elements
const firstDiv = document.getElementById('firstDiv');
const secondDivContainer = document.getElementById('secondDivContainer');

// Add click event listener to the first div
firstDiv.addEventListener('click', function() {
    // Create the second div
    const secondDiv = document.createElement('div');
    secondDiv.className = 'second-div';
    secondDiv.innerHTML = `
        
Second Div

        
I was created by the first div!

        
Click the reset button to remove me.

    `;
    
    // Add the second div to the container
    secondDivContainer.appendChild(secondDiv);
    
    // Show the second div with animation
    setTimeout(() => {
        secondDiv.style.display = 'block';
    }, 10);
});

// Reset button functionality
function resetSecondDiv() {
    secondDivContainer.innerHTML = '';
}
